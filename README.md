# Packet Tracer - Configure Access Control



## Overview


This project demonstrates the configuration of authentication, authorization, secure wireless access, email services, and FTP services using Cisco Packet Tracer.


The lab focuses on practical cybersecurity and network administration skills commonly used in Security Operations Center (SOC), IT support, and enterprise environments.


---



# Technologies & Concepts Used



- Cisco Packet Tracer
- AAA Authentication
- RADIUS Server Configuration
- WPA2 Wireless Security
- SMTP & POP3 Email Services
- FTP Services
- User Authentication & Authorization
- DHCP Configuration
- Windows Command Prompt
- Basic Access Control
- Network Service Security



---



# Project Objectives



## Part 1 — AAA Authentication


- Configured AAA services on a RADIUS server
- Created secure user credentials
- Configured WPA2 wireless authentication
- Connected wireless clients using authenticated access
- Verified DHCP IP assignment



---


## Part 2 — Email Services


- Enabled SMTP and POP3 services
- Configured enterprise email accounts
- Connected email clients
- Sent and received emails between users
- Verified secure mail communication



---



## Part 3 — FTP Services


- Enabled FTP services
- Created FTP users with different privilege levels
- Uploaded and downloaded files using FTP
- Tested user permissions
- Verified access control restrictions
